GitHub使用简记

GitHub使用简记

建立连接

使用ssh，将本地电脑与Github账户连接起来。

fork

fork需要运行、更新的repository，可以复制一份到自己的repositories。

clone

将自己的repository clone到本地，进行二次开发。

上传

add、commit、push，将本地的代码发布到自己GitHub中的repository。

pull

申请将自己的repository与原本的repository合并。
